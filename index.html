What Powers This Pipeline

🐙

GitHub

Source of truth for your code. Branch-based workflow with webhook integration to trigger Jenkins on every push to main.

⚙️

Jenkins

Declarative Jenkinsfile pipeline with stages for build, test, approval gate, and deploy. Hosted on the same EC2 instance.

☁️

AWS EC2

Ubuntu server running Jenkins and Nginx. Files deployed directly via SSH copy. Auto-reloads on every approved deployment.

✋

Approval Gate

Jenkins input() step halts the pipeline. A named approver clicks Deploy or Abort before anything touches production.

🔒

Security

GitHub tokens stored as Jenkins credentials. Sudoers rules limit Jenkins user permissions to only what's needed for deployment.

🔔

Notifications

Post-pipeline hooks send email or Slack messages on success or failure. Know exactly what happened and when.
